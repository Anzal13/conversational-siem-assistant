# Conversational SIEM Assistant

This repository contains the implementation of a Conversational SIEM Assistant for
security log analysis, threat investigation, and automated incident reporting using
Natural Language Processing (NLP).

## Features
- Security log ingestion and preprocessing
- Log-based anomaly detection using ML
- NLP-based query understanding
- Automated threat investigation and reporting
- Conversational interface for analysts

## Technology Stack
- Backend: Python (FastAPI / Flask)
- NLP: Transformer-based models (BERT)
- Frontend: Web-based conversational UI
- Data: Public security log datasets
